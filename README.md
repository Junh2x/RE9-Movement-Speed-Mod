# Better Movement Speed - RE9 Requiem

A movement speed modifier mod for Resident Evil 9 Requiem (Biohazard Requiem).

## Download

**[Nexus Mods](https://www.nexusmods.com/residentevilrequiem/mods/82?tab=description)**

## Features

- Separate walk/run speed multipliers
- Per-character settings (Grace / Leon)
- In-game UI for real-time adjustment (REFramework menu → `[Better Movement Speed]`)
- Settings auto-saved to JSON

## Requirements

- [REFramework](https://www.nexusmods.com/residentevilrequiem/mods/9)

## Installation

1. Install REFramework
2. Copy `re9_movement_speed.lua` to `<game folder>/reframework/autorun/`

## Configuration

Press `Insert` in-game to open the REFramework menu, then find `[Better Movement Speed]`.

| Setting | Range | Default |
|---------|-------|---------|
| Walk Speed (Grace) | 0.5 - 3.0 | 1.3 |
| Run Speed (Grace) | 0.5 - 3.0 | 1.3 |
| Walk Speed (Leon) | 0.5 - 3.0 | 1.3 |
| Run Speed (Leon) | 0.5 - 3.0 | 1.3 |
